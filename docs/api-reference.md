# API Reference

This page documents the complete public `pyshed` API — every class, method, and
function exported from the package. It is generated automatically from the type
stubs shipped with pyshed (`pyshed/__init__.pyi`), so the signatures here always
match the installed version. Prose is thin where the stubs are signature-only and
is being filled in over time.

For a task-oriented tour, start with the [Quickstart](quickstart.md) and the
[Staged API](guide/staged-api.md) guide.
